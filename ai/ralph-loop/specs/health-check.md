## 요구사항

헬스체크 엔드포인트가 서버 상태와 가동 시간을 반환한다.

## 세부 사항

- `GET /health`로 접근 가능
- 응답 형식: JSON
- 응답 필드: `status`, `uptime`
- 서버 시작 시점부터의 가동 시간을 초 단위로 반환

## 수락 기준

- [ ] GET /health가 200 상태코드를 반환한다
- [ ] status 필드가 "healthy"이다
- [ ] uptime 필드가 비어있지 않다
